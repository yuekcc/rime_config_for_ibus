# RIME for ibus

参考文档：

- https://github.com/rime/home/wiki/UserData
- https://github.com/rime/home/wiki/RimeWithSchemata#rime-with-schemata


## 安装

- `dnf install ibus-rime`
- 重启 ibus
- 添加中文，rime 输入法
- 将 yaml 文件复制到 `$HOME/.config/ibus/rime`
- 切换到 rime 输入法，点击菜单 -> 部署
